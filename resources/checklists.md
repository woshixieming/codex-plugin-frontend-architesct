# FAOS Checklists

## Accessibility

- Semantic HTML first.
- Keyboard accessible interactive elements.
- Labels for form controls.
- Visible focus.
- Dialog focus trap and restore.
- Reduced motion respected.

## Responsive

- Avoid fixed widths.
- Prevent horizontal overflow.
- Support mobile, tablet, desktop.
- Handle dense tables.
- Avoid hover-only behavior.

## Token

- No raw colors.
- No raw spacing.
- No raw typography.
- No raw radius.
- Prefer semantic tokens.
