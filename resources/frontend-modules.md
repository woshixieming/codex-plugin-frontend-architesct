# FAOS Module Notes

## React
Use functional components, typed props, composition, and forwardRef where needed.

## Vue 3
Prefer SFC, script setup, typed props, emits, and slots.

## Vue 2
Use Options API, props validation, slots, and computed classes.

## Tailwind
Prefer theme tokens. Avoid arbitrary values. Extract repeated class groups.

## Ant Design
Prefer library components. Customize through theme tokens before CSS overrides.

## Element Plus
Prefer library components. Customize through CSS variables or theme config.

## Dashboard
Use AppShell, Sidebar, TopBar, FilterToolbar, DataTable, MetricCard, EmptyState, ErrorState.

## Mobile H5
Handle touch targets, safe-area, keyboard overlap, viewport height, and overflow.
