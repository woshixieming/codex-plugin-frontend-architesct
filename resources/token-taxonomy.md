# Token Taxonomy

## Primitive Tokens

Raw design values.

Examples:

- color.blue.600
- spacing.4
- radius.md

## Semantic Tokens

Meaning-based tokens.

Examples:

- color.action.primary.bg
- color.surface.default
- color.text.muted

## Component Tokens

Component-specific tokens.

Examples:

- button.primary.bg
- card.border
- input.focus.ring

## Recommended Flow

primitive -> semantic -> component
