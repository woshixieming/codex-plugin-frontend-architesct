# Architecture

FAOS is organized into four layers.

## Core

Stable skills used by every project.

## Modules

Framework, styling, UI-library, and domain-specific rules.

## Templates

Reusable output templates.

## Resources

Reference checklists and design knowledge.

## Rule Priority

1. User instruction
2. Project convention
3. UI library module
4. Framework module
5. Styling module
6. Domain module
7. Core rule
