# Usage

## Main Prompt

```text
Use Frontend Architect OS.
Analyze project context, select modules, infer the UI pattern, generate the implementation, and run a design review.
```

## Generate Component

```text
Use FAOS to generate a reusable UserCard component using existing project conventions.
Use tokens, include accessibility, responsive behavior, usage examples, and a review checklist.
```

## Review UI

```text
Use FAOS to review the current frontend changes for token usage, component reuse, accessibility, responsive behavior, documentation drift, and Storybook coverage.
```

## Migrate Legacy UI

```text
Use FAOS to audit the current UI, extract token candidates, propose component consolidation, and create an incremental migration plan.
```
