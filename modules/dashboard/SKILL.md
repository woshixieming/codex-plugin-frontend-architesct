# Dashboard Module

## Purpose

Apply dashboard, admin, CRM, ERP, and management system patterns.

## Common Patterns

- AppShell
- Sidebar
- TopBar
- Breadcrumb
- FilterToolbar
- DataTable
- MetricCard
- ChartCard
- DetailPanel
- EmptyState
- ErrorState

## Rules

- Compose pages from reusable blocks.
- Keep filter, table, pagination, and bulk action patterns consistent.
- Include loading, empty, and error states.
- Handle table overflow and dense layouts carefully.
