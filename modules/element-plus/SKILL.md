# Element Plus Module

## Purpose

Apply Element Plus / Element UI integration rules.

## Rules

- Prefer library components for common UI.
- Customize using CSS variables or theme configuration.
- Avoid deep overrides unless necessary.
- Use wrappers only for domain-specific abstraction.
- Preserve built-in behavior.

## Review Checks

- duplicate library components
- excessive style overrides
- inconsistent token usage
- broken form accessibility
