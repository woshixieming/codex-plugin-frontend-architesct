# Vue 3 Module

## Purpose

Apply Vue 3 / Nuxt 3 frontend architecture conventions.

## Rules

- Prefer Single File Components.
- Prefer `<script setup lang="ts">` if TypeScript is detected.
- Use `defineProps` and `defineEmits`.
- Use slots for composition.
- Use computed classes for variants.
- Keep props minimal and typed.

## Review Checks

- prop definitions
- emits declarations
- slot usage
- scoped style consistency
- accessibility
- token usage
