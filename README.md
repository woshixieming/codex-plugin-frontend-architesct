# Frontend Architect OS

A Codex plugin for frontend architecture and design-system workflows.

## Codex-compatible structure

```text
frontend-architect-os/
├── .codex-plugin/
│   └── plugin.json
├── skills/
│   ├── architect-orchestrator/
│   │   └── SKILL.md
│   ├── component-generator/
│   │   └── SKILL.md
│   └── ...
├── resources/
├── templates/
└── docs/
```

## Main usage

```text
@frontend-architect-os Use FAOS to analyze project context, select modules, infer the UI pattern, generate implementation, and run a design review.
```

## Direct skill usage

```text
@frontend-architect-os/component-generator Generate a reusable UserCard component.
```
